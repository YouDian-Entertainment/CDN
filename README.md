# CDN

第三方存储管理工具。

## 环境

Vue 2 + Vue-Router + Vuex + Electron@7 + Webpack 4

## 项目依赖环境

- nodejs: v10.10.0
- electron: v7.1.2
- vue: v2.6.10
- vuex: v3.1.1
- vue-router: v3.0.6

## 开发

```bash
# 拉取项目
git clone https://github.com/YouDian-Entertainment/CDN.git
# 安装依赖
cd CDN
npm install
# 运行开发环境
npm run dev
```
